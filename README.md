# 藍星科技空中課堂直播網站

這是一個可直接公開部署的直播網站範本，依照企業直播觀看頁常見結構重做，並加入老師端螢幕分享與學員端同步觀看功能。

## 已包含功能

- 上方頻道列、登入狀態、直播提示
- 直播標題、人氣、二維碼分享、簽到、有獎答題
- 大型播放器與播放控制列
- 老師端螢幕分享開播
- 學員端輸入教室碼加入觀看
- 複製學員觀看連結
- 互動聊天室、簡介、匿名提問、回放頁籤
- 手機版單欄觀看介面

## 使用方式

1. 老師打開公開網址。
2. 點「老師開播」。
3. 瀏覽器跳出分享視窗後，選擇要展示的螢幕、視窗或分頁。
4. 複製教室碼或學員連結給學員。
5. 學員打開同一個網站，輸入教室碼，或直接打開學員連結觀看。

## 注意

目前版本使用瀏覽器 WebRTC 做即時螢幕分享，適合小班測試與內部課程。若要大量學員穩定觀看，建議後續接入正式直播串流服務或自架訊號伺服器。
