# X 學院直播網站復刻版

這是一個可直接開啟的靜態直播網站範本，依照企業直播頁常見結構重做：

- 上方白色頻道列、登入狀態、直播提示
- 標題區、人氣、二維碼分享、簽到、有獎答題
- 左側大播放器與播放控制列
- 右側掃碼觀看與廣告資訊欄
- 下方互動聊天室、簡介、匿名提問、回放頁籤
- 手機版自動改成單欄版面

## 使用方式

直接用瀏覽器開啟 `index.html` 即可預覽。

可替換內容：

- `assets/live-cover.jpeg`：直播封面
- `index.html` 內的品牌名稱、直播標題、課程文案、分享網址
- `script.js` 內的互動範例與複製連結
- 後續可接 HLS 直播串流、登入、WebSocket 聊天、簽到與答題後台
